Why did I receive this error message?

If the Microsoft service known as Background Intelligent Transfer Service (BITS) is disabled, it may be interfering with your ability to install updates. This service allows the transfer of files between a client and server.

To enable BITS

  1. In Windows XP, click Start, Control panel.
  2. In Category view, click the Performance and maintenance icon. If you are in Classic view, proceed to step 3.
  3. Double-click the Administrative tools icon.
  4. Double-click the Services icon.
  5. Double-click the Background Intelligent Transfer Service item from the Services list.
  6. In the Background Intelligent Transfer Service properties dialog box, choose Manual or Automatic from the Startup type list box.
